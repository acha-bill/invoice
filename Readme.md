# Invoice

A simple API for invoice generation and tracking

## Requirements
1. MongoDB
2. Spring

## Features
1. jwt based auth
2. create and email invoices
3. Track paid/unpaid invoices
4. View charts and graphs

## Running the app
1. `mvn clean install`
2. `mvn spring-boot:run`

## Documentation
1. Run the app
2. Visist `locahost:8080/swagger-ui.html`
